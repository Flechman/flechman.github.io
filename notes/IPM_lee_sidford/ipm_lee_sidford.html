# IPM Part 2: The Lee-Sidford Barrier

  • These notes are based on my semester research project in the Theoretical Computer Science Lab at EPFL, where I was supervised by Prof. Dr. Kapralov and Kshiteej Sheth.
  • To understand the context in which the objects presented here are used, read my notes on Interior Point Methods.

# 1. Warm-Up: The Logarithmic Barrier

One simple barrier that one can consider for linear constraints is the logarithmic barrier :

Definition 1: Logarithmic barrier - Let the n-dimentional polytope with m constraints K={Ax≥b}=∩i∈[m]​{ai⊤​x≥bi​}. The logarithmic barrier is defined as

ϕ(x)=−i=1∑m​ln(ai⊤​x−bi​)

To study its self-concordance, we start by proving the following lemma

Lemma 1: If ϕ1​ and ϕ2​ are ν1​- and ν2​-self-concordant barriers on K1​ and K2​ respectively, then ϕ1​+ϕ2​ is a ν1​+ν2​-self-concordant barrier on K1​∩K2​.

Proof. The two properties of ν-self-concordance (Definition 4) are validated by just combining the inequalities of self-concordance of ϕ1​ and ϕ2​ (and using triangle inequality for the second property).


Lemma 2: The Logarithmic barrier ϕ(x)=−∑i=1m​ln(ai⊤​x−bi​) is m-self-concordant.


Proof. Take any i∈[m] and ϕi​(x)=−ln(ai⊤​x−bi​). Let si​=ai⊤​x−bi​.

Dϕi​(x)[v]=⟨∇ϕi​(x),v⟩=∇ϕi​(x)⊤v=−si​ai⊤​v​

and we just proved the first property of self-concordance for ϕi​(x).
Now,

∇ϕi​(x)=−si​ai​​∇2ϕi​(x)=si2​ai​ai⊤​​

∥∇ϕi​(x)∥∇2ϕi​(x)−12​=∥∇2ϕi​(x)−1∇ϕi​(x)∥∇2ϕi​(x)2​​=∇ϕi​(x)⊤∇2ϕi​(x)−1∇ϕi​(x)=si​ai⊤​​si2​(ai​ai⊤​)−1si​ai​​=ai⊤​(ai​ai⊤​)−1ai​=1​

Since ϕ(x)=∑i=1m​ϕi​(x) and by Lemma 1., ϕ(x) is m-self-concordant.

From our IPM framework, we conclude that using the logarithmic barrier yields an O(m​) iteration algorithm for solving Linear Programs. The main problem with this is that since m can get exponential in n, this can get very inefficient, for example when the LP has a lot of repeated (or very similar) constraints. The next logical step is to try and reweigh the constraints, and give less weight to the ones that are repeated.

# 2. The Weighted Logarithmic Barrier

We are looking for a barrier that behaves like −∑i=1m​wi​ln(ai⊤​x−bi​) for wi​ to be determined later. We thus define ψ(x)=−∑i=1m​wi​ln(ai⊤​x−bi​) and we will study its self concordance.

Lemma 3: We have

∇ψ(x)⊤(∇2ψ(x))−1∇ψ(x)≤i∑​wi​

And

D3ψ(x)[h,h,h]≤2imax​​wi​σi​(W​Ax​)​​(D2ψ(x)[h,h])23​

where σi​(W​Ax​) is the leverage score of Ax​W​, namely σi​(W​Ax​)=(W​Ax​(Ax⊤​WAx​)−1Ax⊤​W​)i,i​

Proof. We have that ∇ψ(x)=Ax⊤​w and ∇2ψ(x)=Ax⊤​WAx​. Hence,

∇ψ(x)⊤(∇2ψ(x))−1∇ψ(x)=w​⊤(W​Ax​(Ax⊤​WAx​)−1Ax⊤​W​)w​

But W​Ax​(Ax⊤​WAx​)−1Ax⊤​W​ is an orthogonal projection matrix, so its operator norm is smaller than 1. We can then conclude that

∇ψ(x)⊤(∇2ψ(x))−1∇ψ(x)≤w​⊤w​≤i∑​wi​

For the second part of the proof, let si​=ai⊤​x−bi​ be the ith slack condition. We have

D3ψ(x)[h,h,h]=−2i∑​wi​(si​ai⊤​h​)3≤2imax​​si​ai⊤​h​​i∑​wi​(si​ai⊤​h​)2

Furthermore,

​si​ai⊤​h​​=​si​ai⊤​(Ax⊤​WAx​)−21​(Ax⊤​WAx​)21​h​​≤(Ax​(Ax⊤​WAx​)−1Ax⊤​)i,i​∥h∥∇2ψ(x)​

Hence, we have

D3ψ(x)[h,h,h]≤2imax​​wi​σi​(W​Ax​)​​(D2ψ(x)[h,h])23​

Rescaling ψ by a constant, we get that

D3ψ(x)[h,h,h]≤2i∑​wi​imax​​wi​σi​(W​Ax​)​​(D2ψ(x)[h,h])23​

Which gives us a self-concordance factor of ∑i​wi​maxi​​wi​σi​(W​Ax​)​​. It depends on x, but if wi​=σi​(W​Ax​), note that we have that ∑i​wi​maxi​​wi​σi​(W​Ax​)​​=∑i​σi​(W​Ax​)≤n because of the properties of leverage scores.

Hence with this intuition, we can see how one can try to look for a O(n) self-concordant barrier. The next section will present the actual barrier found by Lee and Sidford, and give some details about it.

# 3. The Lee-Sidford Barrier

As seen in the previous section, we want to weight the ith constraint with a weight wi​ such that wi​=σi​(W​Ax​). This corresponds exactly to the definition of the ℓ∞​ Lewis weights. By their recursive nature, it's very hard to compute them exactly, so one can try to relax those condition to taking w to be the vector of the ℓp​ Lewis weights, with p large enough.

Lee and Sidford introduced the following barrier :

ϕp​(x)=wi​≥0max​logdet(Ax⊤​W1−p2​Ax​)−(1−q2​)i=1∑m​wi​

And proved the following result :


Theorem 1: If A has full rank, then for any p>0, ϕp​ is a O(nmp+11​) self concordant barrier. In particular, for p=Θ(logm),ϕp​ is a O(nlogO(1)m) self concordant barrier.


The proof is lengthy and can be found in Reference 1, but one insight that might be interesting is the following (proof omitted):

Lemma 4: We have

Ax⊤​Σp​(x)Ax​⪯∇2ϕp​(x)⪯(1+p)Ax⊤​Σp​(x)Ax​

Where Σp​(x)i,i​ is the ith ℓp​ Lewis weight of Ax​.

This lemma is insightful because it shows that the hessian of ϕp​ is spectrally close to Ax⊤​Σp​(x)Ax​, which is the Hessian of the ℓp​ Lewis weights reweighted logarithmic barrier, and this can justify why the Lee-Sidford barrier behaves as wanted.

# 4. Putting it all together

We have exhibited a O~(n) self-concordant barrier, which can be used to get a O~(n​log(1/ϵ) iteration algorithm to solve Linear Programs. One important thing to study now is the per iteration cost, in particular the cost to compute the barrier. The Lee-Sidford is very costly to compute because it involves determinants, and ℓp​ Lewis weights. Not being careful about how one computes it might ruin all our previous efforts.

The idea is to compute the barrier iteratively: the broad picture is that from ϕp​(x), x, and x′, if x and x′ are close enough, one can estimate ϕp​(x′)−ϕp​(x) with satisfactory precision, because ϕp​ is smooth enough. Furthermore, computing ϕp​(x′) from ϕp​(x) only takes a logarithmic number of linear systems to solve. We can now state the final result of Lee and Sidford :


Theorem 2: Given an interior point x0​ of a Linear Program, there exists an algorithm that outputs a feasible point x such that c⊤x≤OPT+ϵ with constant probability in O~(n​log(1/ϵ)Tw​) time, where Tw​ is the work needed to compute (A⊤DA)−1q for a positive diagonal matrix D and a vector q.


Depending on the nature of the Linear Program, Tw​ might vary, and as such, the most general result with state of the art techniques states that one can solve approximately a Linear Program in time O~((nnz(A)+rank(A)ω)n​log(1/ϵ)) where ω is the constant multiplication matrix.
